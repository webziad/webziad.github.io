آموزش html



0 تومان

اولین قدم شروع طراحی وبسایت، آموزش HTML هست. HTML یه زبان نشانه گذاری هست که با کمک اون میتونی تگ هایی بنویسی تا سایت رو روی اون تگ ها پیاده کنی. همونجوری که هیچ موجودی نمیتونه بدون اسکلت زندگی کنه، هیچ وبسایتی هم نمیتونه بدون HTML لود بشه. با یادگیری HTML خیلی راحت و قشنگ میتونی یه اسکلت سفت و سخت برای وبسایتت درست کنی.
